## 📖 关于 GPT Image 2

GPT Image 2（2026年4月21日发布）是 OpenAI 的新一代图像生成模型，核心亮点：

- **中文渲染零翻车**：复杂版式、长文排版、中英文混合均能精准呈现
- **UI 截图级真实**：App 界面、网页、游戏截图逼真到可商用
- **照片级质感**：光影、透视、材质细节精准还原
- **编辑能力强**：保留面部、修改文字、调整布局

## 🚀 快速开始

1. 打开 [网站](https://xturnvk.github.io/gpt-image2-prompts/)
2. 点击分类标签筛选平台
3. 在搜索框输入关键词
4. 点击展开卡片，复制提示词
5. 粘贴到 ChatGPT / ChatIMG.ai / fal.ai 生成图片
